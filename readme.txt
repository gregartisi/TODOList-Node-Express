Basic Application Node.js to manage a todo list. you can:
-add things to your to do list
edit tasks in the list
-delete things from the list
-the list is shared in real time with other users by websockets

Backend: server.js
front: views/todo.ejs

Node.js at backend with Express.js 
ejs template at frontend.

to make it work, you need to have node js intalled with modules.the modules you need to download on npm are:
express, express-session,body-parser => go to project directory , open console, npm install "module_name"
